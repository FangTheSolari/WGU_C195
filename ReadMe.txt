C195 - Appointment Management Application

The main purpose of this program is to allow users to manage customer appointments via GUI

Author: Adam Rutland-Ruiz
Contact: arutla2@wgu.edu
Application Version 1.0
Date: 7/26/2022

IDE: IntelliJ IDEA Community Edition 2021.1.3
JDK: Java SE 17.0.2
SDK: JavaFX 17.0.2
MySQL Connector Driver Version: MySQL Connector Java 8.0.25

How to use the program:
1. The program will open to a login screen. To access the program use "admin" as the username and password.
2. Once logged in you will go to the main menu screen. Here you can view the list of customers and appointments. The program will alert you if there is an appointment within the next 15 minutes.
3. From the main menu screen you can add, update, and delete customers and appointments.
4. You can view all appointments, or sort by current week/month.
5. You can generate reports by clicking the "Reports" button.
6. By clicking on the various radio buttons you can generate the following reports:
        -the total number of customer appointments by type and month
        -a schedule for each contact in your organization that includes appointment ID, title, type and description, start date and time, end date and time, and customer ID
        -Schedules for all contacts, sorted by user name.
